FAQ

Small answers before you run it.

Does it include a model runtime?

No. It is a focused interface for an existing generation setup and keeps the heavy runtime separate.

Where do generated files go?

They stay with your own setup. J AI Studio reads the outputs and keeps gallery metadata for review.

Does it support video?

Yes, when your installed generation stack exposes compatible video workflows.

How do updates work?

Use Settings -> Update in a Git checkout, or run git pull, npm install, and npm run build.
