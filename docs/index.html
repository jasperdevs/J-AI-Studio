FAQ

Small answers before you install.

Does it replace ComfyUI?

No. ComfyUI remains the runtime, graph editor, node source, and model source of truth.

Is it local?

Yes. J AI Studio connects to your local ComfyUI server and keeps generated files in your local setup.

Does it support video?

Yes, when your ComfyUI install exposes compatible video generation nodes and workflow templates.

How do updates work?

Use Settings -> Update in a Git checkout, or run git pull, npm install, and npm run build.
