FAQ

Useful answers before you run it.

What is it?

J AI Studio is a simple browser UI for ComfyUI. It is for generating and reviewing outputs, not editing node graphs.

What do I need installed?

Node.js 20 or newer, plus a working ComfyUI install. By default it looks for ComfyUI at http://127.0.0.1:8188.

Where do files go?

Generated media stays in your local ComfyUI output setup. J AI Studio keeps gallery metadata in its own data folder.

Does video work?

Yes, if your ComfyUI setup has compatible video nodes and workflows installed.

Can I use my own workflow?

Yes. Export a ComfyUI API workflow, add the J AI Studio control mapping, then import it from Settings.

How do updates work?

Use Settings -> Update in a Git checkout. The manual path is git pull, npm install, npm run build, then restart.

Use ComfyUI without living in the graph.

Keep the node editor for setup. Use J AI Studio for the daily generation loop.
